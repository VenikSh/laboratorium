Часть 1. Генерация массива

ArrGen01. Даны целые числа N и K. Сформировать массив размера N, все элементы которого равны K, и вывести элементы этого массива. При N=10 K=7 => [7,7,7,7,7,7,7,7,7,7]. Генератор ArrFill(n, k).

ArrGen02. Дано целое N. Сформировать массив размера N с элементами 1, 2, 3, 4, … и вывести элементы этого массива. Генератор ArrGen(n, x->T).

ArrGen03. Дано целое N. Сформировать массив размера N с элементами 2, 4, 6, 8, … и вывести элементы этого массива. Генератор ArrGen(n, first, x->T).

С использованием цикла

С использованием генератора

// ArrGen03

var n := ReadLnInteger('n=');

var x := 2;

for var i := 0 to n - 1 do

begin

  a[i] := x;

  Print(a[i]);

end;

Часть 2. Поиск в массиве

ArrFind01. Дан целочисленный массив оценок (от 2 до 5) размера N. Если в массиве есть двойки ИЛИ тройки, то вывести True, иначе вывести False. Генератор ArrRandomInteger(n, 2, 5), метод q.Any(x->Boolean).

Примеры решения:

С использованием цикла

С использованием методов

##

// ArrFind01

var n := ReadLnInteger('n=');

var a := ArrRandomInteger(n, 2, 5);

Print(a);

var res:=false;

for var i := 0 to n - 1 do

  if (a[i]=2) or (a[i]=3) then

  begin

    res:=true;

    break;

  end;

Print(res);

 

##

// ArrFind01_2

var n := ReadLnInteger('n=');

var a := ArrRandomInteger(n, 2, 5);

Print(a);

a.Any(x -> (x = 2) or (x = 3)).Print;

Часть 4. Фильтрация элементов

ArrFilter01. Дан целочисленный массив размера N. Получить массив, содержащий элементы исходного массива, кратные 3. Вывести элементы полученного массива и его размер K. Генератор ArrRandomInteger(n), метод a.FindAll(T->boolean). Процедура SetLength(b, k).

Примеры решения:

С использованием цикла

С использованием методов

##

//  ArrFilter01

var n := ReadLnInteger('n=');

var a := ArrRandomInteger(n);

PrintLn(a);

var b: array of integer;

var k := 0;

for var i := 0 to n - 1 do

  if a[i] mod 3 = 0 then

  begin

    k += 1;

    SetLength(b, k);

    b[k - 1] := a[i];

  end;

Print(b, k);

##

//  ArrFilter01_2

var n := ReadLnInteger('n=');

var a := ArrRandomInteger(n);

PrintLn(a);

var b := a.FindAll(x -> x mod 3 = 0);

Print(b, b.Length);

Часть 7. Срезы массивов

Срез массива – это его подмассив.

Срез массива a имеет вид:

a[from : to : step]

– и содержит копию элементов массива a в диапазоне индексов от from до to с шагом step.

Элемент с индексом from включается в диапазон, а элемент с индексом to – не включается.

 

Если from отсутствует, то предполагается, что from = 0.

Если to отсутствует, то предполагается, что

to = индексу последнего элемента + 1.

Способы получения индекса последнего элемента:

Length(a)-1,       a.High,      ^1.

Если step не указывается, то предполагается, что step = 1.

Шаг step может быть отрицательным.

Если step<0, то пропущенный from полагается равным индексу последнего элемента, а пропущенный to - индексу первого элемента - 1.

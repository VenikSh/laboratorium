1) Двумерные массивы (матрицы): формирование

ZMatrix01. Даны целые положительные числа M и N. Сформировать целочисленную матрицу размера M × N, у которой все элементы I-й строки имеют значение 10·(I + 1) (I = 0, …, M−1).
MatrGen(m, n, (i, j)-> T);
10 10 10…
20 20 20…
30 30 30…
…

Примеры решения:

Путем заполнения в цикле

С использованием генераторов

##

// ZMatrix01

var (m, n) := ReadlnInteger2('m n =');

 

var a := new integer[m, n];

 

for var i := 0 to m - 1 do // по строкам

  for var j := 0 to n - 1 do // по столбцам

    a[i, j] := (i + 1) * 10;

 

a.Print;

##

// ZMatrix01_2

var (m, n) := ReadlnInteger2('m n =');

 

var a := MatrGen(m, n, (i, j)-> (i + 1) * 10);

 

a.Print;

Путем перебора в цикле

ZMatrix31. Дана матрица размера M × N. Найти индексы строки и столбца для элемента матрицы, наиболее близкого к среднему значению всех ее элементов.

Примеры решения:

Путем перебора в цикле

Путем генерирования массива по строкам

##

// ZMatrix31

var (m, n) := ReadlnInteger2('m n =');

 

var a := MatrRandomInteger(m, n, 0, 9);

a.PrintLn;

 

var sum := 0;

for var i := 0 to m - 1 do

  for var j := 0 to n - 1 do

    sum += a[i, j];

 

var avg := sum / (m * n);

PrintLn(avg);

 

var (iMin, jMin) := (0, 0);

for var i := 0 to m - 1 do

  for var j := 0 to n - 1 do

    if (abs(a[i, j] - avg) < abs(a[iMin, jMin] - avg)) then

    begin

      iMin := i;

      JMin := j;

    end;

 

Print(iMin, jMin);

##

// ZMatrix31_2

var (m, n) := ReadlnInteger2('m n =');

 

var a := MatrRandomInteger(m, n, 0, 9);

a.PrintLn;

 

var avg := ArrGen(m, i -> a.Row(i).Average).Average;

avg.Println;  // - среднее значение

 

var b := a.ConvertAll((x, i, j)-> abs(x - avg));

b.PrintLn;    // - матрица разностей a[i,j]-avg

 

var min := ArrGen(m, i -> b.Row(i).Min).Min;

min.Println;  // - минимальная разность

 

b.Indices((x, i, j)-> x = min).Print;

              // - индексы минимальной разности

// или так:

// b.ForEach((x, i, j)-> if x = min then Print(i, j));

ZMatrix32. Дана целочисленная матрица размера M × N. Найти индекс первой из ее строк, содержащих равное количество положительных и отрицательных элементов (нулевые элементы матрицы не учитываются). Если таких строк нет, то вывести −1.

Получить массив количеств положительных по строкам: n1:= ArrGen(m,i->T);  .Row(i);  .Count(x->boolean);
Получить массив количеств отрицательных по строкам: n2:= ArrGen(m,i->T);  .Row(i);  .Count(x->boolean);
Получить массив разностей количеств: r:= n1.Zip(n2, (x,y)->x-y).ToArray;
Получить индексы строк с нулевой разницей: r.Indices(x->x=0)  или: r.IndexOf(0)

значение максимального/миниального элемента в строке:

  var jMax, jMin: integer;

  var max := integer.MinValue;

  var min := integer.MaxValue;

    begin

      JMax := j;

      max := a[i, j];

    end;

    if a[i, j] < min then

    begin

      JMin := j;

      min := a[i, j];

    end;

  a[i, jMin] := max;

ZMatrix94. Дана квадратная матрица порядка M. Обнулить элементы матрицы, лежащие одновременно ниже главной диагонали (включая эту диагональ) и выше побочной диагонали (также включая эту диагональ). Условный оператор не использовать.

ZMatrix95. Дана квадратная матрица порядка M. Обнулить элементы матрицы, лежащие одновременно ниже главной диагонали (включая эту диагональ) и ниже побочной диагонали (также включая эту диагональ). Условный оператор не использовать.

ZMatrix96. Дана квадратная матрица A порядка M. Зеркально отразить ее элементы относительно главной диагонали (при этом элементы главной диагонали останутся на прежнем месте, элемент A0,1 поменяется местами с A1,0, элемент A0,2 — с A2,0 и т. д.). Вспомогательную матрицу не использовать.

ZMatrix97. Дана квадратная матрица A порядка M. Зеркально отразить ее элементы относительно побочной диагонали (при этом элементы побочной диагонали останутся на прежнем месте, элемент A0,0 поменяется местами с AM−1,M−1, элемент A0,1 — с AM−2,M−1 и т. д.). Вспомогательную матрицу не использовать.

ZMatrix98. Дана квадратная матрица A порядка M. Повернуть ее на угол 180 (при этом элемент A0,0 поменяется местами с AM−1,M−1, элемент A0,1 — с AM−1,M−2 и т. д.). Вспомогательную матрицу не использовать.

ZMatrix99. Дана квадратная матрица A порядка M. Повернуть ее на угол 90 в положительном направлении, т. е. против часовой стрелки (при этом элемент A0,0 перейдет в AM−1,0, элемент AM−1,0 — в AM−1,M−1 и т. д.). Вспомогательную матрицу не использовать.

ZMatrix100. Дана квадратная матрица A порядка M. Повернуть ее на угол 90 в отрицательном направлении, т. е. по часовой стрелке (при этом элемент A0,0 перейдет в A0,M−1, элемент A0,M−1 — в AM−1,M−1 и т. д.). Вспомогательную матрицу не использовать.

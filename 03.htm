AInteger01. Дано расстояние L в сантиметрах. Используя операцию деления нацело, найти количество полных метров в нем (1 метр = 100 см). При L=6937 результат: 69

Пример решения:

##

// AInteger01

var L := ReadInteger;

var P := L div 100;

Print(P);

AInteger12. Дано трехзначное число. Вывести число, полученное при прочтении исходного числа справа налево. При исходном значении=247 результат: 742

Пример решения:

##

//  AInteger12

var a:=ReadLnInteger;               // 247

 

var a1 := a div 100;                // 2

var a2 := a div 10 mod 10;          // 4

var a3 := a mod 10;                 // 7

 

var b := a3 * 100 + a2 * 10 + a1;   // 7*100 + 4*10 + 2

 

PrintLn(b);                         // 742

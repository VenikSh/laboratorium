// Proc37

function Power1(a, b: real): real;

begin

  result := if a <= 0 then 0 else exp(b * ln(a));

end;

Proc40. Описать функцию Exp1(x, ε) вещественного типа (параметры x, ε — вещественные, ε > 0), находящую приближенное значение функции exp(x):
exp(x) = 1 + x + x2/(2!) + x3/(3!) + … + xn/(n!) + …
(n! = 1·2·…·n). В сумме учитывать все слагаемые, большие ε. С помощью Exp1 найти приближенное значение экспоненты для данного x при шести данных ε. Пример решения:

// Proc40

function Exp1(x, e: real): real;

begin

  var i := 0;    // номер слагаемого

  var k := 1.0;  // текущее слагаемое

 

  while k >= e do

  begin

    result += k;

    i += 1;

    var fact := 1.0;

    for var j := 2 to i do

      fact *= j;

    k := power(x, i) / fact;

  end;

 

end;

 

BEGIN

  var x := ReadLnReal;

  var e := ReadLnReal;

  Print(Exp1(x, e));

  // для сравнения - стандарт.функция exp(x):

  Print(Exp(x));

END.

Часть 1. Одномерные массивы: формирование

ZArray01. Дано целое число N (> 0). Сформировать и вывести целочисленный массив размера N, содержащий N первых положительных нечетных чисел: 1, 3, 5, … . Генератор ArrGen(n, first, T->T).

Примеры решения:

С использованием цикла

С использованием генератора

##

// ZArray01

var n := ReadLnInteger('n=');

var a := new integer[n];

var x := 1;

for var i := 0 to n - 1 do

begin

  a[i] := x;

  x += 2;

end;

Print(a);

##

// ZArray01_2

var n := ReadLnInteger('n=');

var a := ArrGen(n, 1, x -> x + 2);

Print(a);

ZArray03. Дано целое число N (> 1), а также первый член A и разность D арифметической прогрессии. Сформировать и вывести массив размера N, содержащий N первых членов данной прогрессии: A,    A + D,    A + 2·D,    A + 3·D,    … . Генератор ArrGen(n, integer->T).

ZArray04. Дано целое число N (> 1), а также первый член A и знаменатель D геометрической прогрессии. Сформировать и вывести массив размера N, содержащий N первых членов данной прогрессии: A,    A·D,    A·D2,    A·D3,    … . Генератор ArrGen(n, integer->T).

ZArray05. Дано целое число N (> 2). Сформировать и вывести целочисленный массив размера N, содержащий N первых элементов последовательности чисел Фибоначчи FK: F0 = 1,        F1 = 1,        FK = FK−2 + FK−1,    K = 2, 3, … . ArrGen(n, first, second, (T,T) ->T).

ZArray06. Даны целые числа N (> 2), A и B. Сформировать и вывести целочисленный массив размера N, в котором элемент с индексом 0 равен A, элемент с индексом 1 равен B, а каждый последующий элемент равен сумме всех предыдущих. Вложенный цикл.

ZArray18. Дан массив A ненулевых целых чисел размера 10. Вывести значение первого из тех его элементов AK, которые удовлетворяют неравенству AK < A9. Если таких элементов нет, то вывести 0. Генератор ArrRandomInteger(n), метод a.Find(x->Boolean).

Примеры решения:

С использованием цикла

С использованием методов

##

// ZArray18

var n := ReadLnInteger('n='); // 10

var a := ArrRandomInteger(n);

PrintLn(a);

for var i := 0 to n - 1 do

  if a[i]<a[9] then

  begin

    Print(a[i]);

    break;

  end;

##

// ZArray18_2

var n := ReadLnInteger('n='); // 10

var a := ArrRandomInteger(n);

PrintLn(a);

Print(a.Find(x -> x < a[9]));

Генератор ArrRandomInteger(n), метод a.FindLastIndex(x->Boolean).   При [14,87,78,48,31,57,26,75,56,87]  результат: 8.  При [94,87,63,28,20,45,39,15,85,81]  результат: -1.

С использованием цикла

С использованием кортежного присваивания

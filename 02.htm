ЛР02. Арифметические выражения

Вычислить нижеприведенные выражения.

Рекомендации:

•      Сложные выражения следует разбивать на более простые, присваивая их промежуточным переменным, а потом подставляя в изначальное выражение.

•      Не забывайте ставить * для умножения.

•      Помните о приоритетах выполнения операций; для изменения приоритета используйте скобки.

•      Количество открывающих скобок должно быть равно количеству закрывающих.

•      Нужные математические функции ищите в Справочной системе: Системный модуль PABCSystem – Математические подпрограммы.

•      Если нужной функции нет, вспомните, как она выражается через другие функции.

•      Не забывайте: параметр функции должен быть в скобках.

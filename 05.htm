ЛР05. Условный оператор If

 

Разделы справочной системы:
– Справочник по языку – Операторы – Условный оператор;
– Справочник по языку – Выражения и операции – Условная операция.

 

AIf01. Дано целое число. Если оно является положительным, то вычесть из него 8; в противном случае не изменять его. Вывести полученное число. При исходном значении 12 результат: 4. При исходном значении -6 результат: -6.

AIf02. Дано целое число. Если оно является положительным, то вычесть из него 8; в противном случае прибавить к нему 6. Вывести полученное число. При исходном значении 10 результат: 2. При исходном значении -9 результат: -3.

Пример решения:

С помощью оператора If

С помощью операции If

##

// AIf02

var a := ReadInteger;

var p: integer;

if a > 0 then

  p := a - 8

else

  p := a + 6;

Print(p);

##

// AIf02-2

var a := ReadInteger;

var p := if a > 0 then a - 8 else a + 6;

// или:

// var p := a > 0 ? a - 8 : a + 6;

Print(p);

При исходном значении 10 результат: 2. При исходном значении -5 результат: 1. При исходном значении 0 результат: 10.

AIf21. Даны целочисленные координаты точки на плоскости. Если точка совпадает с началом координат, то вывести 0. AIf28. Дан номер года (положительное целое число). Определить количество дней в этом году, учитывая, что обычный год насчитывает 365 дней, а високосный — 366 дней. Високосным считается год, делящийся на 4, за исключением тех годов, которые делятся на 100 и не делятся на 400 (например, годы 300, 1300 и 1900 не являются високосными, а 1200 и 2000 — являются). При исходном значении 2100 результат: 365. При исходном значении 168 результат: 366.

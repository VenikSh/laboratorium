4) Непрерывный показ файлов двойным щелчком мыши

Сначала разместите на форме таймер – компонент timer1.

Двойным щелчком по timer1 создайте процедуру:

procedure Form1.timer1_Tick(sender: Object; e: EventArgs);

begin

  NumSlide += 1;

  if NumSlide > Slide.High then

    NumSlide := 0;

  pictureBox1.Load(Slide[NumSlide]); 

end;

– благодаря этому смена слайдов будет происходить автоматически с определенным временны́м интервалом. Этот интервал задается свойством таймера Interval и по умолчанию составляет 100 миллисекунд.

По умолчанию таймер не запущен, – это значит, что его свойство Enabled = false.

Запускать таймер будем двойным щелчком мыши. Для этого на форме выберите элемент pictureBox1, в Инспекторе объектов найдите событие MouseDoubleClick и двойным щелчком по нему создайте процедуру обработки двойного клика мыши:

procedure Form1.pictureBox1_MouseDoubleClick(sender: Object; e: MouseEventArgs);

begin

  Timer1.Enabled := not Timer1.Enabled;

end;

– при первом двойном щелчке таймер запускается, а при повторном двойном щелчке – останавливается.

 

Запустите и протестируйте приложение: двойной щелчок мыши запускает непрерывную смену файлов, а повторный двойной щелчок – останавливает.

Очень быстро, правда? Скорость можно регулировать, изменяя значение свойства Interval. Этим сейчас и займемся.

1) Обеспечение рисования мышью на pictureBox

В коде модуля Unit1.pas после ключевого слова Implementation вставьте операторы объявления глобальных переменных:

implementation

var g : Graphics;   // инкапсуляция поверхности рисования

var x, y : integer;                 // координаты мыши

var p := new Pen(Color.Black);      // перо черного цвета

 

Для элемента pictureBox1 создайте процедуру события Paint, которая выполняется при открытии формы и при необходимости перерисовки элемента:

procedure Form1.pictureBox1_Paint(sender: Object; e: PaintEventArgs);

begin

  g := Graphics.FromHwnd(pictureBox1.Handle);

end;

– создается новый объект класса Graphics из дескриптора Handle элемента pictureBox1. Теперь к переменной g можно будет применять методы, рисующие примитивы. Вариант: g := e.Graphics;

 

Процедура события MouseDown для pictureBox1:

procedure Form1.pictureBox1_MouseDown(sender: Object; e: MouseEventArgs);

begin

  (x, y) := (e.X, e.Y);

end;

– запоминает координаты мыши в момент нажатия кнопки мыши (левой или правой), передаваемые через параметр e.

Параметр e:MouseEventArgs обладает свойствами, позволяющими определить: какая была нажата кнопка мыши (свойство e.Button), сколько раз была нажата кнопка (e.Click), на сколько делений прокручено колесико мыши (e.Delta) и координаты мыши (e.X, e.Y).

 

Процедура события MouseMove для pictureBox:

procedure Form1.pictureBox1_MouseMove(sender: Object; e: MouseEventArgs);

begin

    g.DrawLine(p, x, y, e.x, e.y);

    (x, y) := (e.X, e.Y);   

end;

– рисует линию пером p, начиная от прежних значений координат (x, y) к текущим (e.X, e.Y) и затем запоминает текущие координаты.

Пока здесь не анализируется, нажата ли кнопка мыши и какая нажата.

 

Выполните приложение и протестируйте: убедитесь, что рисование происходит непрерывно всего лишь перемещением мыши без нажатия кнопки.

1) Движение pictureBox с помощью таймера timer

В разделе Implementation объявим глобальные переменные dx и dy, опредедяющие приращение и направление движения:

implementation

var (dx, dy) := (1, 1);

 

Разместите на форме таймер timer1.

Таймер позволяет выполнять заданные действия через заданный интервал времени.

Запускается таймер путем установки его свойства Enabled в состояние True. Задайте timer1.Enabled = true.

Свойство timer1.Interval задает интервал времени между событиями Tick; по умолчанию – 100 ms; оставьте как есть.

Действия, какие надо выполнять, задаются в процедуре события Tick; впишите в её тело следующий код:

procedure Form1.timer1_Tick(sender: Object; e: EventArgs);

begin

  pictureBox1.Left := pictureBox1.Left + dx;

  if (pictureBox1.Left + 25 >= Width) or (pictureBox1.Left <= 0) then

    dx := -dx; 

  pictureBox1.Top := pictureBox1.Top + dy;

  if (pictureBox1.Top + 48 >= Height) or (pictureBox1.Top <= 0) then

    dy := -dy; 

end;

Width – ширина формы; Height – высота формы.
Положение элемента pictureBox1 внутри формы задается свойствами:
pictureBox1.Left – расстояние от левого края формы до левой стороны элемента,
pictureBox1.Top – расстояние от верхнего края формы до верхней стороны элемента.
Через каждые 100 ms эти свойства увеличиваются или уменьшаются в зависимости от направления движения, соответственно, на dx или на dy.
Направление движения меняется при достижении pictureBox1 края формы.
Константы 25 и 48 – для учета размеров квадратика и ширины бордюров формы.

 

Выполните приложение и убедитесь: квадратик передвигается и рикошетит от края формы.

1) Программное создание элемента p типа pictureBox

В разделе Implementation объявите переменную p типа PictureBox:

Двойным щелчком по форме создайте процедуру для обработки события Load, возникающего при загрузке формы, и введите операторы, задающие свойства p, и оператор, включающий p в коллекцию элементов формы:

  p.Height := 20;

  p.Left := 50;

end;

 

2) Перемещение p вслед за курсором мыши

Сделаем так, чтобы квадратик догонял курсор мыши.

Для этого для создайте процедуру события MouseMove – перемещение курсора мыши:

procedure Form1.Form1_MouseMove(sender: Object; e: MouseEventArgs);

begin

  if e.X > p.Left then

    p.Left += 1

  else

    p.Left -= 1;

 

  if e.y > p.Top then

    p.Top += 1

  else

    p.Top -= 1;   

end;

Здесь e.x и e.y это текущие координаты курсора мыши, они передаются в процедуру через параметр e.

p.Left и p.Top – смещение p относительно левого и, соответственно, верхнего края формы.

 

3) «Убегание» p от щелчка мыши

Сделаем так, чтобы квадратик убегал от щелчка.

Для этого для создайте процедуру события MouseClick – щелчок мыши:

procedure Form1.Form1_MouseClick(sender: Object; e: MouseEventArgs);

begin

  if e.X > p.Left then

    p.Left -= 100

  else

    p.Left += 100;

 

  if e.y > p.Top then

    p.Top -= 100

  else

procedure Form1.Form1_MouseClick(sender: Object; e: MouseEventArgs);

begin

  var k := Length(a) + 1;

    var g := PABCSystem.Random(0, 255);

    var b := PABCSystem.Random(0, 255);

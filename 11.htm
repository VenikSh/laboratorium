Примеры решения:

С использованием цикла

С использованием последовательностей

##

// ASeries01

var n := ReadLnInteger('n=');

var k := 0;

 

loop n do

begin

  var a := ReadInteger;

  if a.IsEven then

    k += 1;

end;

 

Print(k);

##

// ASeries01_2

var q := ReadSeqInteger(n);

Часть 3. Число как последовательность цифр

ASeries31. Дано целое положительное число N. Сколько в его записи нечетных цифр? При N=3591670  результат:  5. Функция ReadInteger, методы n.ToString, q.Count(x->boolean), функция StrToInt(x), метод x.IsOdd.

Примеры решения:

С использованием цикла

Путем превращения n в строку (строка это последовательность символов)

##

// ASeries31

var n := ReadLnInteger('n=');

var k := 0;

 

repeat

  var a := n mod 10; // последняя цифра

  if a.IsOdd then    // если нечетная

    k += 1;

  n := n div 10      // отбросить последнюю цифру

until n = 0;

 

Print(k);

##

// ASeries31_2

var n := ReadLnInteger('n=');

Print(n.ToString.Count(x -> StrToInt(x).IsOdd));

«Дан символ» – означает: ввод с клавиатуры: var c:=ReadLnChar;     или присвоение константы: var c:='a';

«Дана строка» – означает: ввод с клавиатуры: var s:=ReadLnString;     или присвоение константы: var s:='строка';

 

String01. Дан символ C. Вывести его код (т. е. номер в кодовой таблице). c:=ReadLnChar;   c.Code, Ord(c)
C='B' => 66

String02. Дано целое число N (32 ≤ N ≤ 126). Вывести символ с кодом, равным N. n:=ReadLnInteger;   Chr(n)
N=33 => !

String03. Дан символ C. Вывести два символа, первый из которых предшествует символу C в кодовой таблице, а второй следует за символом C. c:=ReadLnChar;   Pred(c), Succ(c)
C='B' => A C

String04. Дано целое число N (1 ≤ N ≤ 26). Вывести N первых прописных (т. е. заглавных) букв латинского алфавита. n:=ReadLnInteger;   c.Code, Ord(c), Chr(n);     ('A'..'Z').Take(5)
N=5 => abcde

Примеры решения:

Путем посимвольного перебора в цикле

С использованием методов

##

// String04

var n := ReadlnInteger;

 

for var i := ord('A') to ord('A') + n - 1 do

  print(chr(i));

##

// String04-2

var n := ReadlnInteger;

 

'ABCDEFGHIJKLMNOPQRSTUVWXYZ'.Take(n).PrintLn;

 

('A'..'Z').Take(n).Println;

String05. Дано целое число N (1 ≤ N ≤ 26). Вывести N последних строчных (т. е. маленьких) букв латинского алфавита в обратном порядке (начиная с буквы «z»). c.Code, Ord(c), Chr(n);     ('a'..'z').TakeLast(5)
N=5 => vwxyz

String06. Дан символ C, изображающий цифру или букву (латинскую или русскую). Если C изображает цифру, то вывести строку «digit», если латинскую букву — вывести строку «lat», если русскую — вывести строку «rus». c.isDigit, c.isLetter;    (c in ['A'..'Z', 'a'..'z', 'А'.. 'Я', 'а'.. 'я', 'Ё', 'ё'])
C='9' => 'digit';     C='Z' => 'lat';     C='Я' => 'rus'

S='qwert' => 113 116

String08. Дано целое число N (> 0) и символ C. Вывести строку длины N, которая состоит из символов C. c+c, StringOfChar(c, n)
N=6 C='z' => S='zzzzzz'

String09. Дано четное число N (> 0) и символы C1 и C2. Вывести строку длины N, которая состоит из чередующихся символов C1 и C2, начиная с C1. c+c, s+s
N=6 C1='z' C2='v' => S='zvzvzv'

s:='qwErЩщTYfжЖUgh' => 8

String17. Дана строка. Преобразовать в ней все строчные буквы (как латинские, так и русские) в прописные. Length(s), s[i], s[i].toUpper;   s.toUpper
s:='QWERTЙЦУК' => s='qwertйцук'

String18. Дана строка. Преобразовать в ней все строчные буквы (как латинские, так и русские) в прописные, а прописные — в строчные. Length(s), s[i], s[i].isUpper, s[i].toUpper, s[i].isLower, s[i].toLower;    s.ToCharArray.ConvertAll(x->x.IsLower?x.toUpper:x.toLower)

String19. Дана строка. Если она представляет собой запись целого числа, то вывести 1, если вещественного (с дробной частью) — вывести 2; если строку нельзя преобразовать в число, то вывести 0. Считать, что дробная часть вещественного числа отделяется от его целой части десятичной точкой «.».Length(s), s[i].isDigit; s.TryToInteger(n), s.TryToReal(n)
S='12345' => 1;     S='123.45' => 2;     S='qwert' => 0

4+7−2−8=1

String25. Дана строка, изображающая десятичную запись целого положительного числа. Вывести строку, изображающую двоичную запись этого же числа. 5921 = 5000+900+20+1 = 5·103 + 9·102 + 2·101 + 1·100

 

String26. Дано целое число N (> 0) и строка S. Преобразовать строку S в строку длины N следующим образом: если длина строки S больше N, то отбросить первые символы, если длина строки S меньше N, то в ее начало добавить символы «.» (точка). Length(s), s[i], s.PadLeft(n,'.')

String27. Даны целые положительные числа N1 и N2 и строки S1 и S2. Получить из этих строк новую строку, содержащую первые N1 символов строки S1 и последние N2 символов строки S2 (в указанном порядке). Length(s), s[i], s1.Left(n1), s2.Right(n2)

Создадим три варианта приложений – три простейших калькулятора, различающихся используемыми элементами управления, размещаемыми на форме.

Для создания приложения создается проект, представляющий из себя совокупность файлов в отдельной папке:

-      файл проекта Project1.pabcproj,

-      файл главной программы Project1.pas,

-      файлы модулей форм Unit1.pas, Unit2.pas…

-       и служебные файлы.

Как правило, имя папки совпадает с именем проекта. После компиляции и сборки получается одноименный выполняемый файл Project1.exe.

Создадим процедуру-обработчик события Click для кнопки button1  +

Для этого щелкните по ней двойным щелчком. В результате в коде модуля Unit1.pas будет вписан код процедуры с «пустым» телом.

Введите следующие операторы:

procedure Form1.button1_Click(sender: Object; e: EventArgs);

begin

  var a:= StrToFloat(textBox1.Text);

  var b:= StrToFloat(textBox2.Text);

  var c := a + b;

  textBox3.Text := FloatToStr(c); 

end;

Поскольку значение свойства Text имеет тип String, его надо преобразовать к типу вещественного числа Float (Real); это делается с помощью функции StrToFloat. А потом – наоборот: полученный результат надо преобразовать к строковому типу – с помощью функции FloatToStr.

 

Нажмите F9 и протестируйте кнопку  +

Закройте приложение.

Вернитесь в Дизайнер и создайте так же процедуры-обработчики клика для кнопок –, * , / . Сами догадайтесь, чем они отличаются от процедуры для кнопки + .

 

Для кнопки button5 (Выход) процедура-обрабочик клика:

procedure Form1.button5_Click(sender: Object; e: EventArgs);

end;

– содержит оператор вызова процедуры Close, закрывающей окно приложения.

Создайте процедуру-обработчик клика для кнопки Вычислить:

procedure Form1.button1_Click(sender: Object; e: EventArgs);

begin

  var a := StrToFloat(textBox1.Text);

  var b := StrToFloat(textBox2.Text);

  var c: real;

  case comboBox1.SelectedIndex of

    0: c := a + b;

    1: c := a - b;

    2: c := a * b;

    3: c := a / b;   

  end;

  textBox3.Text := FloatToStr(c);  

end;

Здесь анализируется свойство SelectedIndex элемента управления comboBox1, оно содержит номер выбранной строки списка операций, нумерация элементов списка начинается с 0.

 

Процедура-обработчик клика для кнопки Выход должна содержать оператор Close.

Для groupBox, label, button и для формы Form1 задайте значения свойства Text.

Для формы задайте свойство StartPosition = CenterScreen.

Для всех полей textBox задайте свойство TextAlign = Right.

Для textBox3 (результат) задайте свойство ReadOnly = True.

 

Создайте процедуру-обработчик клика для кнопки Вычислить:

procedure Form1.button1_Click(sender: Object; e: EventArgs);

begin

  var a := StrToFloat(textBox1.Text);

  var b := StrToFloat(textBox2.Text);

  var c: real;

  if radioButton1.Checked then

    c := a + b;

  if radioButton2.Checked then

    c := a - b;

  if radioButton3.Checked then

    c := a * b;

  if radioButton4.Checked then 

    c := a / b;   

  textBox3.Text := FloatToStr(c);  

end;

Здесь проверяется свойство Checked радиокнопки radioButton: если оно равно true, значит, данная радиокнопка выбрана.

 

Процедура-обработчик клика для кнопки Выход должна содержать оператор Close.

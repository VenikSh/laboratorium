ЛР08-2. Циклы с условием While и Repeat

 

Разделы справочной системы:
– Справочник по языку – Операторы – Операторы цикла while и repeat;
– Справочник по языку – Операторы – Составной оператор.

 

While01. Даны положительные числа A и B (A > B). На отрезке длины A размещено максимально возможное количество отрезков длины B (без наложений). Не используя операции умножения и деления, найти длину незанятой части отрезка A. При A=62.57  B=14.53  результат: 4.45.

Пример решения:

##

// While01

var (a, b) := ReadReal2;

var p := a - b;

 

while p >= b do

  p -= b;

 

Print(p);

While15. Начальный вклад в банке равен 1000 руб. В конце каждого месяца размер вклада увеличивается на P процентов от имеющейся суммы (P — вещественное число, 0 < P < 25). По данному P определить, через сколько месяцев размер вклада превысит 1100 руб., и вывести найденное количество месяцев K (целое число) и итоговый размер вклада S (вещественное число). While16. Спортсмен-лыжник начал тренировки, пробежав в первый день 10 км. Каждый следующий день он увеличивал длину пробега на P процентов от пробега предыдущего дня (P — вещественное, 0 < P < 50). По данному P определить, после какого дня суммарный пробег лыжника за все дни превысит 200 км, и вывести найденное количество дней K (целое) и суммарный пробег S (вещественное число). При P=37.00 результат: K=7  сумма: 217.790.

 

While17. Дано целое число N (> 0). Используя операции деления нацело и взятия остатка от деления, вывести все его цифры, начиная с самой правой (разряда единиц). При N=951  результат: 1  5  9.

Пример решения:

##

// While17

var n := ReadInteger;

var p: integer;

 

repeat

  p := n mod 10;       // p - последняя цифра в n

  Print(p);

  n := n div 10;       // отсечь последнюю цифру в n

until n = 0;

While23. Даны целые положительные числа A и B. Найти их наибольший общий делитель (НОД), используя алгоритм Евклида:    НОД(A, B) = НОД(B, A mod B),    если B ≠ 0;        НОД(A, 0) = A,
где «mod» обозначает операцию взятия остатка от деления.
При A=105  B=150  результат: 15.

 

While24. Дано целое число N (> 1). Последовательность чисел Фибоначчи Fi определяется следующим образом:    F1 = 1,        F2 = 1,        Fi = Fi−2 + Fi−1,    i = 3, 4, … .  
(1  1  2  3  5  8  13  21...)
Проверить, является ли число N числом Фибоначчи. Если является, то вывести true, если нет — вывести false.
При N=21  результат: True. При N=7  результат: False.

Пример решения:

##

// While24

var n := ReadInteger;

var (a, b) := (1, 1);

 

repeat

  (a, b) := (b, a + b);

until (n = a) or (n < a);

 

Print(n = a);

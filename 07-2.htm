For13. Дано целое число N (> 0). Найти значение выражения 1.1 − 1.2 + 1.3 − … (N слагаемых, знаки чередуются). Условный оператор не использовать. При N=6  результат: -0.3.    При N=7  результат: 1.4.

Пример решения:

Без условного оператора If

С операцией If

##

// For13

var n := ReadInteger;

var s := 0.0;

var z := -1;

 

for var i := 1 to n do

begin

  z *= (-1);

  s += z * (1.0 + 0.1 * i);

end;

 

Print(s);

##

// For13

var n := ReadInteger;

var s := 0.0;

 

for var i := 1 to n do

  s += (if odd(i) then 1 else -1) * (1.0 + 0.1 * i);

 

Print(s);

For36. Даны целые положительные числа N и K. Найти сумму    1K + 2K + … + NK.    Чтобы избежать целочисленного переполнения, вычислять слагаемые этой суммы с помощью вещественной переменной и выводить результат как вещественное число. При  N=5   K=2  результат: 55.00.

Пример решения:

##

// For36

var (n, k) := ReadInteger2;

var s := 0.0;

 

for var i := 1 to n do

begin

  var p := 1.0;

  for var j := 1 to k do

    p *= i;

  s += p;

end;

 

Print(s);

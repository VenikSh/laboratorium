ALoopFor01. Даны целые числа K и N (N > 0). Вывести N раз число K. При К=9 N=5 результат: 9 9 9 9 9.

Примеры решения:

С помощью оператора Loop

С помощью оператора For

##

// ALoopFor01

var (k, n) := ReadInteger2;

loop n do

  Print(k);

##

// ALoopFor01

var (k, n) := ReadInteger2;

for var i := 1 to n do

  Print(k);

ALoopFor13. Дано целое положительное число N. Вывести первые N чисел арифметический прогрессии 1, 3, 5, 7, …, используя цикл loop. При N=5  результат: 1 3 5 7 9.

Пример решения:

##

var n := ReadInteger;

var p := 1;

loop n do

begin

  Print(p);

  p += 2

end;

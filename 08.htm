Арифметическая прогрессия

AWhile01. Дано целое положительное число N. Вывести первые N чисел арифметический прогрессии 1, 3, 5, 7, …, используя цикл while или repeat. При N=7 результат: 1  3  5  7  9  11  13.

Пример решения:

С помощью while

С помощью repeat

##

// AWhile01

var n := ReadInteger;

var p := 1;

var i := 1;

while i <= n do

begin

  Print(p);

  p += 2;

  i += 1

end;

##

// AWhile01

var n := ReadInteger;

var p := 1;

var i := 1;

repeat

  Print(p);

  p += 2;

  i += 1

until i > n;

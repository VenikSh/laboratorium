Вопрос 1. Ответ с использованием текстового поля

На форме выберите tabPage1 – пространство вкладки Вопрос1 и разместите следующие элементы, изменив их свойство Text:

Процедура для кнопки Ответить:

procedure Form1.button1_Click(sender: Object; e: EventArgs);

begin

  if textBox1.Text = '4' then

  begin

    label2.Text := 'Правильно';

    mark +=1;

  end

  else

    label2.Text := 'Неправильно';

  button1.Enabled := False;

end;

– анализирует содержимое поля с ответом textBox1 и в зависимости от него фиксирует результат в надписи label2. И потом, в любом случае, делает кнопку Ответить button1 недоступной, задав её свойству Enabled значение false. В случае правильного ответа количество баллов mark увеличивается на 1.

Оценка. Получение результата

На форме выберите tabPage4 – пространство вкладки Оценка и разместите следующие элементы, изменив их свойство Text:

label8
 

label7
 

button4
 

button5
 

 

Процедура для кнопки Получить оценку:

procedure Form1.button4_Click(sender: Object; e: EventArgs);

begin

  label7.Text := 'Набрано баллов: ' + mark;

  case mark of

    3: label8.Text := 'Оценка: 5 (Отлично)';

    2: label8.Text := 'Оценка: 4 (Хорошо)';

    1: label8.Text := 'Оценка: 3 (Плохо)';

    0: label8.Text := 'Оценка: 2 (Неуд)';

  end;

end;

– В надпись label7 заносится количество набранных баллов mark. И в зависимости от mark рассчитывается оценка и фиксируется в надписи label8.

 

Процедура для кнопки Пройти тест заново:

procedure Form1.button5_Click(sender: Object; e: EventArgs);

begin

  button1.Enabled := true;

  button2.Enabled := true;

  button3.Enabled := true;

  label2.Text := '';

  label4.Text := '';

  label6.Text := '';

  label7.Text := '';

  label8.Text := '';

  textBox1.Text := '';

  radioButton1.Checked := false;

  radioButton2.Checked := false;

  radioButton3.Checked := false;

  radioButton4.Checked := false;

  checkBox1.Checked := false;

  checkBox2.Checked := false;

  checkBox3.Checked := false; 

  mark := 0; 

  tabControl1.SelectedIndex := 0;

end;

Все элементы переводятся в начальное состояние:
кнопки становятся доступными,
надписи Правильно/Неправильно очищаются,
текстовое поле тоже очищается,
все радиокнопки и флажки переводятся в невыбранное состояние,
количество баллов обнуляется
и осуществляется переход на первую вкладку (индексация вкладок – с 0).

Задание. Создать приложение для анкетирования

В папке своей группы создайте проект Anketa.

На форме должно быть три вкладки по категориям вопросов: ФИО, Образование, Работа, четвертая вкладка – Сводная.

Вкладка ФИО должна содержать надписи с вопросами и текстовые поля для ответов: Фамилия, Имя, Отчество, Год рождения, Адрес.

Вкладка Образование должна содержать:
1) Группбокс Имею диплом (аттестат) и четыре флажка для ответа: Общее среднее, Начальное профессиональное, Среднее профессиональное, Высшее.
2) Надпись Учебное заведение и текстовое поле для ответа.
3) Надпись Год окончания и текстовое поле для ответа.

Вкладка Работа должна содержать надписи с вопросами и текстовые поля для ответов:
1) Группбокс Род деятельности предприятия и радиокнопки для ответа: Производство, Торговля, Услуги, Сельское хозяйство.
2) Надпись Название предприятия и текстовое поле для ответа.
3) Надпись Должность и текстовое поле для ответа.

Вкладка Сводная должна содержать кнопку Свести информацию и многострочное текстовое поле richTextBox1. По нажатию кнопки данные со всех вкладок должны быть сведены в это текстовое поле – построчно, в удобочитаемой форме.

На вкладке Сводная разместите еще одну кнопку Записать в файл и поле для указания в нем имени файла.
Указание. Используйте оператор:
richTextBox1.SaveFile(textBox10.Text);
где textBox10 – поле с именем файла (номер, возможно, будет другой).

 

Примечание. Вместо текстового поля типа richTextBox можно использовать поле типа textBox (со свойством MultiLine = True).

В отличие от richTextBox у поля textBox отсутствуют методы SaveFile и LoadFile.

Сохранить содержимое поля textBox1.Text в файле с именем NFile можно так:

System.IO.File.WriteAllText(NFile, textBox11.Text);

А, наоборот, прочитать файл и поместить его содержимое в поле можно так:

textBox1.Text := System.IO.File.ReadAllText(NFile);

Имя файла NFile можно брать как свойство диалогов:

NFile := openFileDialog1.FileName;

NFile := saveFileDialog1.FileName;
